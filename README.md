# NÃO PENSA AMIGAO
Bot de discord que simula o Soundpad, permitindo salvar áudios do youtube como comandos de fácil utilização.

### Comandos:

Salva áudio do youtube como comando:
```
!add {nome_do_comando} {url_do_youtube}
```
Toca o áudio do youtube:
```
!{nome_do_comando}
```
